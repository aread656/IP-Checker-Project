# IP-Checker-Project
A university coursework project from December 2025. Showcases use of web development in HTML, CSS, and JS, with backend functionality using Java, Python, C++, PHP, and Javascript. CI pipelining and Docker container use also showcased.
To run this project in its current state (January 2026), please run "docker compose down -v" followed by "docker compose up --build" from the root folder. The program will then run on port 80, accessible through "localhost".
